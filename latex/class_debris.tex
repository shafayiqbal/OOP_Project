\doxysubsection{Detailed Description}
This class creates the debris when a \mbox{\hyperlink{class_vehicle}{Vehicle}} is destroyed. 

Whenever a \mbox{\hyperlink{class_car}{Car}} collides with another \mbox{\hyperlink{class_car}{Car}} or with \mbox{\hyperlink{class_sheep}{Sheep}} an explosion animation is displayed, this class handles the animation and drawing of the explosion.
