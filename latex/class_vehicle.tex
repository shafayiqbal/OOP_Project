\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_vehicle}{Vehicle}} class, the parent for all vehicles. 

This class is inherited by \mbox{\hyperlink{class_bike}{Bike}}, \mbox{\hyperlink{class_car}{Car}}, Racecar, \mbox{\hyperlink{class_truck}{Truck}} and \mbox{\hyperlink{class_truck2}{Truck2}}. It provides all the functionality relating to the movement of the cars and their speed.
